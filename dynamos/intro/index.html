The Framework

RUA — The Engine Under the Hood

Every DynApp runs on RUA — Kee Lee's meta-cognitive learning framework. Recognize. Understand. Apply.

R

Recognize

Decompose the learning target. BEEF: Background → Evidence (CHOP) → Explanation (SOOI) → Feedback.

U

Understand

Deep comprehension. 3C: Collocation → Contexts → Connotation. Know what it pairs with, where it lives, what it implies.

A

Apply

Deliberate practice. GAP: Gauge the gap → Analyze the cause → Practice, fail, correct, repeat.

MFP
Evaluation (Meaning·Form·Pronunciation)

Platform Roadmap

From Demo to Operating System

Every phase validates before the next commits. Gates at every stage.

Now · Week 1

Push Engine Demo

Single-page builder. Prove the AI can generate useful teaching tools from natural language intent. ⏸️ Validation gate with real teachers.

Weeks 2–4

Vertical Slice — UK Study Abroad

One complete learner path. Intention → Diagnosis → Missions → Tasks → Teacher Session → Passport. Real students. Real feedback.

Month 2–3

DynaSaurus Migration

Swap chatbot memory for competence graph. Add persistent learner profiles. MFP/CCC/GAP evaluation engine. Keep the UI — replace the core.

Month 4–6

Teacher SaaS + Marketplace

Tiered pricing. DynApp marketplace opens. SrKeeda's network of 40+ instructors onboarded. AI matchmaking v1.

Month 7–12

Full DynamOS

Multi-intention types. Push/Pull engine live. Cross-domain competence tracking. Competence Passport as portable credential.
