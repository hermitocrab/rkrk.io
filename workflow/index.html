DAY 1

试卷五维解码

看清一张卷子到底考什么

📥 你给它一份试卷的内容（拍照、打字、OCR 都行）
📤 它给你《试卷五维解码表》
试卷导入→题目拆解→考点标注→五维归类 
你是一位专业的英语试卷分析专家。请从五个维度分析这份试卷：

1. 知识点维度 — 列出每道题考查的语法点和词汇点，标注题号
2. 认知层级维度 — 每道题属于：识记 / 理解 / 应用 / 分析
3. 题型维度 — 归类：选择题 / 填空题 / 匹配题 / 判断题 / 简答题 / 作文
4. 难度维度 — 每道题评：简单 / 中等 / 困难，简要说明理由
5. 分值分布维度 — 按听说读写语法词汇六大板块归类，统计各板块分值占比

分析完成后，请指出：
- 哪些考点权重过大（远高于真实考试的比例）
- 哪些考点缺失（试卷没考到但考试会考的）
- 这对教学优先级的启示

最后输出一份《试卷五维解码表》，包含以上五个维度的完整分析，以及一个总览框：总题数、总分值、各技能覆盖率、最重要的一条发现。

DAY 2

班级学情筛查

看清班级共性漏洞

📥 你给它每个学生每道小题的得分
📤 它给你《班级漏洞筛子图》
小题得分→维度统计→筛子图定位→问题排序 
你是一位数据驱动的英语教学分析师。我会给你每个学生每道小题的得分，请帮我做以下分析：

1. 维度统计 — 按听力、阅读、写作、语法、词汇等技能板块，统计每个板块的班级平均分、最低分、分差

2. 筛子图定位 — 把错题分成三类：
   · 大洞——全班 80% 以上都错了，共性漏洞，优先解决
   · 中漏——50%–80% 错了，常见薄弱点
   · 小滴——不到 50% 错了，个别问题

3. 问题排序 — 综合考虑三个因素排出优先级：
   · 丢分多少（影响面大的排前面）
   · 好不好改（短期内能快速提升的排前面）
   · 前置条件（有些问题必须先解决，才能解决其他问题）

4. 用文字描述一张热力图：
   · 红色区域：紧急，全班性问题
   · 黄色区域：中等，超过一半学生有问题
   · 绿色区域：良好，大多数学生已掌握

输出一份《班级漏洞筛子图》。

DAY 3

学生错因归类

看清每个学生为什么错

📥 你给它学生姓名 + 错题 + 学生答案 + 正确答案
📤 它给你《学生错因画像表》
错题导入→AI诊断→五维标签→个体画像 
你是一位诊断型英语教学专家。请分析每个学生的错题，找出错误背后的真正原因——不只是"哪题错了"，而是"为什么错"。

诊断时使用以下五类标签：

1. 知识缺失 — 根本没学过这个知识点
2. 理解偏差 — 学过但用错了（母语干扰、过度类推等）
3. 粗心失误 — 压力下注意力不集中导致的失误
4. 策略错误 — 用了错误的解题方法
5. 心理因素 — 紧张、不自信导致的失分（纠结、改答案等）

为每个学生画一份错因画像：
- 他 / 她最主要的错误类型是什么？（看整体规律）
- 什么情况下最容易出错？（时间压力？特定题型？）
- 有没有明显的母语（中文→英文）迁移痕迹？
- 针对他 / 她的错误类型，应该用什么方式干预？给具体建议

输出一份《学生错因画像表》，每个学生一个区块。

学生错题如下：
【粘贴：学生姓名 + 错题原文 + 学生答案 + 正确答案】
DAY 4

ABC分层任务

同一个考点，不同学生走不同的路

📥 你给它一个知识点 + 学生分层情况
📤 它给你《ABC分层任务单》
学情分层→A/B/C任务设计→难度路径生成 
你是一位分层教学设计专家。针对同一个知识点，请设计三套不同难度层次的学习任务。

知识点：【比如：现在完成时 vs 一般过去时】

A 层 · 基础层（60 分以下的学生）
- 用中文辅助解释这个语法点的核心概念
- 设计辨认型练习（在句子中认出这个时态）
- 提供带词库的填空题
- 达标标准：能 80% 准确地辨认该时态

B 层 · 应用层（60–80 分的学生）
- 设计情境选择题（根据时间状语判断用哪个时态）
- 句子改写练习（一般过去时→现在完成时）
- 改错练习，每题附解释
- 达标标准：能在控制练习中正确使用两个时态

C 层 · 拓展层（80 分以上的学生）
- 开放式写作（写一段话，自然切换两个时态）
- 用自己的话讲清楚每个时态为什么用在这里
- 实战改错（修改一篇中国学生写的、时态混用的作文）
- 达标标准：能自我纠错并解释原因

每层都要包含：
- 3 个阶梯式练习，附答案
- 中国学生容易踩的坑（针对这个知识点的母语干扰提醒）
- 做完后自查是否达标的检查点

输出一份《ABC分层任务单》，一层一个板块。
DAY 5

精准补偿训练

哪里不会，就练哪里

📥 你给它一个具体薄弱点 + 学生当前水平
📤 它给你《个人补偿训练包》
薄弱点→题型匹配→变式训练→答案解析 
你是一位精准补救训练设计师。针对一个具体薄弱点，帮我出一套训练材料。

1. 先确认问题：这个薄弱点到底是什么？用一句话说清楚。

2. 出 3 道诊断题——直接暴露这个薄弱点的题目，题型要和真实考试一致。

3. 出 10 道变式训练题，同一个知识点，换不同题型和角度来练：
   填空 → 选择 → 改错 → 句子改写 → 自由表达
   难度从低到高：先建立信心，再逐步加难度，最后达到考试水平

4. 每道题的错误选项都要附解析：
   - 不只说"答案选 B"，要解释为什么有人会选 A / C / D，选了说明什么理解偏差
   - 特别标注：中国学生在这个知识点上最容易犯什么错

输出一份《个人补偿训练包》，包含完整题目和答案详解。

学生的薄弱点：【具体描述，最好附上他之前的错题】
学生当前水平：【比如初一 / 高一 / 雅思 5.5】
DAY 6

错题复现过关

用间隔复现确认学生真的学会了

📥 你给它学生错题记录（题目 + 日期 + 错误类型）
📤 它给你《错题复现与二次过关表》
错题本→第1/3/7天复现→二次检测→过关标记 
你是一位间隔复习与掌握度验证设计师。用"1-3-7 复现法"帮学生确认他们真的掌握了错题。

第 1 天检测——确认听懂了。出 3 道完全同类型的题
第 3 天检测——确认记住了。出 3 道同知识点但换形式的题
第 7 天检测——确认内化了。出 2 道混合在综合练习里的题

针对错题本里的每一道错题，生成三套检测题。

评分规则：
- 通过：3 对 2（第 7 天 2 对 1 即可）
- 不通过 → 生成一份 3 分钟的微型重教稿，针对这个错误点再讲一遍
- 通过 → 标记已过关，放入复习维持队列

输出一份《错题复现与二次过关表》，列清楚每道错题的三轮检测题、通过 / 未通过栏，以及未通过的补救方案。

DAY 7

个人提分路径

综合所有信息，画出下一步的路线图

📥 你给它前 6 天所有输出 + 学生目标 + 兴趣爱好
📤 它给你《学生个人提分路径卡》
诊断结果→训练记录→AI推荐→下一步计划 
你是一位英语学习路径规划师。综合之前所有的诊断和训练数据，为这个学生画一条清晰的提分路线。

1. 当前画像——用三句话概括：最强的是什么？最弱的是什么？最主要的错误类型是什么？经过前几天的补救训练，有哪些已经改善了？

2. 最大杠杆点——在最短时间内，做哪一件事能让分数提升最明显？基于前面的差距分析来回答。

3. 四周路线图：
   第 1–2 周：优先补最高的短板。每天一个微习惯 + 每周一次检测
   第 3–4 周：巩固离下一个分数段最近的技能。每天一个微习惯 + 每周一次检测

4. 今天就做——给一个具体任务，30 分钟内能完成。不能说"多练听力"，要说"做剑桥 10 Test2 Section1，做完对照原文标出所有没听清的地方"

5. 别浪费时间在——列出哪些对这个学生不重要、可以暂时放一放的事情

6. 动力钩子——把这份计划和学生的兴趣、个人目标挂钩。如果他喜欢篮球，用篮球打比方

7. 成功信号——这个学生怎么知道自己可以进入下一阶段了？给一个可测量的标准

检查一下：Day 4–6 的补救训练有没有改变学生的薄弱点？如果有变化，路线要相应调整。别推荐学生去练一个已经补好的弱项。

输出一页《学生个人提分路径卡》，包含：
📊 当前画像 ｜ 🎯 下一个目标 ｜ 🗺️ 四周路线 ｜ ⚡ 今天就做 ｜ ⚠️ 别浪费时间 ｜ 🏆 成功信号

前 6 天的所有输出摘要如下：

学生的目标分数和时间：
【比如：雅思 6.5，3 个月后考试】
