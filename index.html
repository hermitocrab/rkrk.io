Framework
RUA

A meta-cognitive operating system for learning in the AI era. Every tool I build runs on RUA — Recognise, Understand, Apply.

R

Recognise

Decompose the learning target. BEEF: Background → Evidence → Explanation → Feedback.

U

Understand

Deep comprehension. 3C: Collocation → Contexts → Connotation. Know what it pairs with, where it lives, what it implies.

A

Apply

Deliberate practice. GAP: Gauge the gap → Analyze the cause → Practice, fail, correct, repeat.
