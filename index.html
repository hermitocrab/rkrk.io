The Framework

RUA — Recognise · Understand · Apply

A meta-cognitive operating system for learning in the AI era.

R

Recognise

Decompose the learning target into bite-sized chunks.

Flip for more

🍖 BEEF — Goal Decomposition

  • Background — when, where, who
  • Evidence · CHOP — Caught eye? Highlight? Optimised? Purpose?
  • Explanation · SOOI — Subjective, Objective, Outlying, Intrinsic
  • Feedback — record the emotion as a signal
U

Understand

Deep comprehension before application.

Flip for more

🧩 3C — Deep Comprehension

  • Collocation — what does this naturally pair with?
  • Contexts — in what situations does this appear?
  • Connotation — what's the unspoken meaning?
A

Apply

Systematic practice with feedback loops.

Flip for more

🎯 GAP — Deliberate Practice

  • Gauge — where exactly is the gap?
  • Analyse — trace back to a Recognise chunk
  • Practice — drill, fail, correct, repeat
